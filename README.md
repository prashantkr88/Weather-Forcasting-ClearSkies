# 🌦️ Weather Web Application

A responsive Weather Web Application built using **HTML, CSS, and JavaScript** that allows users to check weather reports, plan activities based on weather conditions, and access useful weather tools.

---

## 🚀 Features

- 🌤️ Check current weather information
- 📊 Detailed weather reports
- 📅 Weather planner for activities
- 🛠️ Weather-related tools
- 📱 Responsive and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Weather API

---
